灵敏度严格按照∂E/∂θ = ( -1+2α-α^2 ) * ε0^T * (∂C/∂θ) * ε0 + α^2 * σ0^T * (∂S/∂θ) * σ0 + (2α^2-2α) * ε0^T * (∂C/∂θ) * σ0
其中，S是柔度矩阵（C的逆），σ0是当前应力，ε0是当前应变。
需要实现这个公式，并设置α=0.5来计算其中包括三项能量贡献：
1.当前应变能变化项：(-1 + 2α - α²) * ε₀ᵀ * (∂C/∂θ) * ε₀
2.当前应力能变化项：α² * σ₀ᵀ * (∂S/∂θ) * σ₀
3.交叉能量项：(2α² - 2α) * ε₀ᵀ * (∂C/∂θ) * σ₀
将三项能量贡献加权求和得到最终灵敏度

确保纤维角度计算公式为：θ_e = π/2 + atan(∂φ/∂y / ∂φ/∂x)

速度场按下述方法进行节点聚合和归一化
步骤1：节点灵敏度聚合
将单元灵敏度加权平均到节点：
∂E/∂φ_i = Σ[∂E/∂θ_e * ∂θ_e/∂φ_i]
需要实现∂θ_e/∂φ_i的计算：∂θₑ/∂φᵢ = [ (∂Nᵢ/∂y)(∂Nᵢ/∂x·φᵢ + Pᵢ) - (∂Nᵢ/∂x)(∂Nᵢ/∂y·φᵢ + Qᵢ) ] / [ (∂Nᵢ/∂x·φᵢ + Pᵢ)² · ( ((∂Nᵢ/∂y·φᵢ + Qᵢ)/(∂Nᵢ/∂x·φᵢ + Pᵢ))² + 1 ) ]
其中θₑ = π/2 + atan(∂φ/∂y / ∂φ/∂x)；∂φ/∂x = Σ(∂Nᵢ/∂x · φᵢ)，∂φ/∂y = Σ(∂Nᵢ/∂y · φᵢ)；Pᵢ = Σ(∂Nₚ/∂x · φₚ)，p ≠ i；Qᵢ = Σ(∂Nq/∂y · φq)，q ≠ i
步骤2：梯度模计算
计算水平集函数的梯度模：|∇φ| = √[(∂φ/∂x)² + (∂φ/∂y)²]
步骤3：速度场构建
速度场 = 节点灵敏度 × 梯度模：v = (∂E/∂φ_i) × |∇φ|
步骤4：自适应时间步长
根据最大灵敏度限制时间步长：Δt = Δθ_max / max|∂E/∂φ_i|
Δθ_max设置为5°（论文值）

水平集更新方程使用标准的迎风差分格式，根据速度场符号选择向前或向后差分
